Automation Capstone

Scaled automation model based on a real industrial material-handling process

For my mechatronics capstone, our team is designing and building a scaled model of an industrial stock car system and developing the control logic needed to automate its operation. Project Overview

The capstone centers on automating a stock car process used in industrial material handling. In the full-scale system, the car moves along a track, positions under specific material bins, receives a target sequence, and then returns to a dump location as part of the charging workflow. Our capstone translates those ideas into a scaled model that demonstrates the motion control and automation logic in a more accessible prototype format.

The engineering challenge is not just making the model move. It is making the system behave predictably under sequence commands, maintain awareness of position, identify valid operating states, and respond safely to faults or interruptions.

What the System Demonstrates

  • Automatic and manual operating modes
  • Position tracking along the system travel path
  • Defined home position behavior
  • Targeting of the correct loading location or process state
  • Sequence-driven movement and operation
  • Emergency stop and permissive logic concepts
  • Sensor-based response to obstructions or invalid conditions
  • Monitoring of communication-dependent system states

Engineering Focus

The value of this project is in how closely it mirrors real automation thinking: define operating modes, understand where the machine is, know what action comes next, verify that conditions are safe, and then execute the correct sequence. That means the project is as much about control architecture and systems thinking as it is about the physical model itself.

Even in a scaled format, the project reflects real-world automation concerns like communication status, state transitions, interlocks, and consistent sequence execution.

My Contributions

My work on the project includes helping translate the real process into a scaled engineering prototype, contributing to the automation concept, and supporting the control logic and integration side of the system. That includes thinking through how the model should move, how states should be tracked, how the control flow should sequence, and how the prototype can clearly demonstrate industrial automation concepts in a physical build.

Because the public version stays high level, I present this project as evidence of my experience with electromechanical systems, automation planning, controls-focused problem solving, and adapting real industrial requirements into a working prototype.

Controls and Safety Concepts

  • Manual override path for non-automatic operation
  • Emergency stop logic built into permissive behavior
  • Home-state logic for known system recovery
  • Fault response when communication conditions are not valid
  • Stop-and-resume handling for object detection scenarios
  • State-based output control rather than uncontrolled motion
  • Prototype design intended to reflect industrial control priorities

Capstone Development Approach

1. Process Translation Break down the real system into portfolio-safe mechanical, sequencing, and controls concepts that can be represented in a scaled model.
2. Scaled Model Design Design the physical structure and motion concept so the prototype can clearly demonstrate travel, positioning, and operation states.
3. Control Logic Planning Define system states, transitions, interlocks, and sequence behavior for safe movement and repeatable operation.
4. Integration and Validation Connect the mechanical model, control hardware, and sensing concepts into a system that demonstrates the intended automation workflow.

Project Summary

This capstone represents the kind of engineering work I am most interested in: taking a real process, breaking it into control problems, building a physical system around it, and using automation logic to make the prototype behave in a structured and reliable way. Even without sharing confidential details, the project highlights systems thinking, electromechanical design, and industrial automation experience.
