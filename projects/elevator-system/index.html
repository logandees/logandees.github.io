Project overview

This elevator was built as a working mechanical and controls project, but the design was also set up to be extendable. The frame can be lengthened with a longer lead screw and additional side supports so more floors can be added without redesigning the whole structure.

Modular frame design

Side supports, rail sections, and stacked floor spacing make the system easier to scale upward in future revisions.

Lead-screw vertical motion

The carriage is driven vertically through a screw-based lift mechanism suited to controlled floor-to-floor movement.

Integrated door concept

The project includes a door system and servo gear components that visually communicate the full elevator behavior, not just lift travel.

Physical + digital documentation

Prototype photos, CAD drawings, exploded views, and an interactive GLB model present both the final result and the engineering process behind it.
