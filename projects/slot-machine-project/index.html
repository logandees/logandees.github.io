Short n’ Sweet
Slots

Built with Python (Pygame) and adapted into a JavaScript web demo

This project started as a fully custom slot machine game built in Python with Pygame. I designed the reel logic, paylines, scatter-triggered free spins, lock-in bonus behavior, asset handling, and presentation system.

To make it portfolio-ready, I also created a browser version so recruiters and employers can interact with the project directly on my website without needing to run the original desktop build.

Features

  • 5x3 reel slot engine with animated spin behavior
  • Multiple paylines and line-win evaluation
  • Scatter-triggered free spins
  • Lock-in style bonus with expanding block logic
  • Custom UI, symbols, sounds, and transition video support
  • Web adaptation built for portfolio presentation

What I Built

I handled the logic side and presentation side of the project: game states, win detection, feature triggering, reel behavior, balance and bet systems, and the process of adapting a desktop Python game into something that can be demonstrated online.

This project shows both software logic and interface design, while also showing that I can package work into a format that is accessible and presentable for real-world review.

Project Summary

This project reflects the kind of work I enjoy most: combining technical logic, interactive design, iteration, and presentation into something that people can actually use. It began as a Python game and was later rebuilt for the browser so it could become a stronger portfolio piece.
